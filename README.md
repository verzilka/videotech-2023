Глаз на сферу натяну: воспроизведение сферического видео в браузере (Videotech 2023)
---
Не забываем установить зависимости
```npm i```.
После установки автоматически запустится скрипт ```prepare```
и сделает в нужных местах ссылки на папку с источником видео.
___
Пакет организован как монореп с 4 основными модулями:
- **video-360-threejs** - пример отображения сферического видео с помощью Three.js
- **webgl-basic-example** - пример отрисовки примитива в WebGL
- **video-360-webgl** - пример отображения сферического видео в ручном режиме с помощью шейдеров
- **video-360-stereoscopic** - пример отображения стереоскопического варианта просмотра сферического видео на Three.js
___
Примеры написаны в первую очередь для понимания принципов работы WebGL и отображения с помощью его сферического видео
___
В каждом модуле предусмотрено 2 команды:
- ```start``` - чтобы запустить пример и посмотреть в браузере
- ```build``` - чтобы собрать javascript и прикинуть характеристики бандла
